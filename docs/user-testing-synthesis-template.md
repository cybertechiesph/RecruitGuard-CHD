**Severity scale:**
- **Bug** — system doesn't work as intended
- **High** — multiple users got stuck; blocks core task
- **Medium** — friction that slowed users but didn't block
- **Low** — polish, edge cases, single-user observations

**Effort scale:**
- **S** — under an hour (copy change, CSS tweak)
- **M** — a day (template restructure, new flow)
- **L** — multi-day (significant rework or backend involvement)
